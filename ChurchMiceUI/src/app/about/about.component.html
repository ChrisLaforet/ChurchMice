About Church Mice

ChurchMice is an open source project that brings a critical component to small churches, an inexpensive to produce online directory. After all, church mice are very social and so they want to be able to recognize other church mice. This software is all about being able to add demographic information and photos to create an online directory for your congregation.

For a very small church, this package also permits adding some basic webpages so that ChurchMice can run the site instead of being an add-on to an existing website.

ChurchMice source code can be downloaded or forked from GitHub at github.com/ChrisLaforet/ChurchMice and can be modified and extended to meet your needs. The code is developed in C#/.NET Core for the services, Angular for the UI/UX, and uses SQL Server DB (currently).
